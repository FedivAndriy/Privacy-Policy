Privacy Policy

Data Collection: This application does not collect, store, or share any personal user data.

Authentication: Authentication is handled exclusively through TikTok’s secure OAuth 2.0 process. The application only accesses the necessary permissions (such as video.upload) to perform its intended functions.

Data Usage: No personal information, videos, or other user data are stored on any external servers. All data processing occurs locally on the user's machine.

Third-Party Services: This application interacts with TikTok’s official API. Please refer to TikTok’s Privacy Policy for information regarding how TikTok handles your data.

Contact: If you have any questions, please contact the developer via the official repository.
